# currency-converter
I have made this app in angular to learn basics and get comfortable with the environment.

### :bangbang: Disclaimer:
Foreign exchange rates API is used to get latest currency rate. `https://api.exchangeratesapi.io/latest`

###  :book: concepts summary

  * Components
 
  * Routing
  
  * Data Binding
  
  * Event Binding
  
  * Structural Directives
  
  * HttpClient Module
